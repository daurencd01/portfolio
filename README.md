# Portfolio Website - Жамеледен Нурадил

This is a minimalist, futuristic portfolio website designed with an "anti-gravity" aesthetic.

## Features
- **Responsive Design**: Optimized for desktop and mobile.
- **Animations**: Subtle floating elements, fade-in on scroll, glassmorphism effects.
- **Structure**: Hero, About, Experience, Leadership, Focus, Contact.

## How to Run

### Option 1: Double-click
1. Open the folder `c:\Портофолия`.
2. Double-click `index.html` to open it in your browser.

### Option 2: Local Server (Recommended for best experience)
1. Open a terminal (PowerShell or Command Prompt).
2. Navigate to the project folder:
   ```bash
   cd c:\Портофолия
   ```
3. Start a Python HTTP server:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and go to `http://localhost:8000`.

## Customization
- **Images**: Place your own images in `assets/images/`. Refer to them in `index.html` using `src="assets/images/filename.png"`.
- **Content**: Edit `index.html` to update text.
- **Styles**: Edit `css/style.css` to tweak colors or animations.
